كلمات المفردات والمرادفات والمضادات (الدروس 1-6)

secure / (Synonyms: safe, protected) / (Antonyms: insecure, unsafe)

آمن (المرادف: safe / protected، المضاد: insecure / unsafe)

strong / (Synonym: powerful) / (Antonym: weak)

قوي (المرادف: powerful، المضاد: weak)

careful / (Synonym: cautious) / (Antonym: careless)

حذر (المرادف: cautious، المضاد: careless)

private / (Synonyms: personal, confidential) / (Antonym: public)

خاص (المرادف: personal / confidential، المضاد: public)

remove / (Synonyms: delete / erase) / (Antonyms: add / keep)

يزيل (المرادف: delete / erase، المضاد: add / keep)

active / (Synonyms: energetic, lively) / (Antonyms: passive, inactive)

نشط (المرادف: energetic / lively، المضاد: passive / inactive)

unsafe / (Synonyms: dangerous, risky) / (Antonyms: safe, secure)

غير آمن (المرادف: dangerous / risky، المضاد: safe / secure)

forever / (Synonyms: always, eternally) / (Antonyms: never, temporarily)

للأبد (المرادف: always / eternally، المضاد: never / temporarily)

kind / (Synonyms: nice, friendly) / (Antonym: unkind)

لطيف (المرادف: nice / friendly، المضاد: unkind)

smart / (Synonyms: intelligent, clever) / (Antonym: stupid)

ذكي (المرادف: intelligent / clever، المضاد: stupid)

trace / (Synonyms: mark, sign) / (Antonyms: loss, disappearance)

أثر / علامة (المرادف: mark / sign، المضاد: loss / disappearance)

trusted / (Synonyms: reliable, dependable) / (Antonyms: untrusted, unreliable)

موثوق (المرادف: reliable / dependable، المضاد: untrusted / unreliable)

adult / (Synonyms: mature, grown-up) / (Antonyms: child, kid)

بالغ / راشد (المرادف: mature / grown-up، المضاد: child / kid)

simple / (Synonym: easy) / (Antonyms: complex, difficult)

بسيط (المرادف: easy، المضاد: complex / difficult)

agree / (Synonym: accept) / (Antonyms: disagree, refuse)

يوافق (المرادف: accept، المضاد: disagree / refuse)

permanent / (Synonyms: lasting, forever) / (Antonym: temporary)

دائم (المرادف: lasting / forever، المضاد: temporary)

scary / (Synonym: frightening) / (Antonym: safe)

مخيف (المرادف: frightening، المضاد: safe)
